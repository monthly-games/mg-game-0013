## 3. 1년차 라이브 플랜

### 3.1 초기 (1~3개월차)

| 주차 | 주요 활동 |
|------|----------|
| 1~2주 | 소프트 런칭, 0006 데이터 기반 설정 |
| 3~6주 | 매칭/밸런스 정책 확립 |
| 7~10주 | 첫 정규 시즌 준비 |
| 11~12주 | 정규 시즌 1 시작 |

**핵심 과제:**
- 0006 데이터 기반 초기 밸런스
- 매칭 시스템 고도화
- 리그 구조 정립

### 3.2 중기 (4~8개월차)

| 월 | 주요 업데이트 |
|----|-------------|
| 4개월 | 시즌 2 + 메타 테마 변경 |
| 5개월 | 상위 리그 전용 연출/스킨 |
| 6개월 | 시즌 3 + 신규 용병 |
| 7개월 | 메타 테마 변경 |
| 8개월 | 시즌 4 + 여름 토너먼트 |

**핵심 과제:**
- 6~8주 단위 시즌 사이클
- 메타 테마 변경 효과 검증
- 상위 리그 콘텐츠 강화

### 3.3 후기 (9~12개월차)

| 월 | 주요 업데이트 |
|----|-------------|
| 9개월 | 토너먼트 시스템 정식 |
| 10개월 | 팀/길드 리그 테스트 |
| 11개월 | 시즌 5 + 챔피언십 예선 |
| 12개월 | 1주년 챔피언십 본선 |

**핵심 과제:**
- 토너먼트 시스템 안정화
- 팀/길드 리그 준비
- 챔피언십 이벤트
