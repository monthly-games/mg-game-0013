## 2. 코어 루프 (Core Loop)

```
┌─────────────────────────────────────────────────────────┐
│  1. 용병 수집/강화                                       │
│         ↓                                               │
│  2. 팀 편성/전략 설정                                    │
│         ↓                                               │
│  3. 리그 매치 참여                                       │
│         ↓                                               │
│  4. 시즌 진행/보상 획득 ─────────────────→ (1로 반복)     │
└─────────────────────────────────────────────────────────┘
```

### 루프 상세 설명

1. **용병 수집/강화**: 다양한 용병 획득 및 육성
2. **팀 편성**: 메타를 고려한 전략적 팀 구성
3. **리그 매치**: 디비전별 리그 경기 참여
4. **시즌 진행**: 시즌 보상 획득, 승강급

## 3. 재미 축 (Fun Pillars)

### 3.1 경쟁/리그 (Competition)
- **목표**: 체계적인 경쟁 구조에서 오는 긴장감과 성취감
- **핵심 경험**: 디비전 승급의 기쁨, 상위 리그 도달 목표
- **주요 요소**:
  - 디비전 시스템 (브론즈→실버→골드→플래티넘→다이아→마스터)
  - 승점 시스템
  - 시즌 보상
  - 명예의 전당

### 3.2 시즌/메타 변화 (Seasonal Meta)
- **목표**: 시즌마다 새로운 전략을 요구하는 신선함
- **핵심 경험**: 이번 시즌은 어떤 메타일까? 하는 기대감
- **주요 요소**:
  - 시즌별 버프/너프
  - 신규 용병 추가
  - 룰 변경
  - 메타 리포트

### 3.3 빌드/카운터 (Counterplay)
- **목표**: 상대를 분석하고 대응하는 전략적 재미
- **핵심 경험**: 카운터 픽으로 역전했을 때의 쾌감
- **주요 요소**:
  - 상성 시스템
  - 상대 정보 스카우팅
  - 밴/픽 시스템
  - 카운터 빌드 가이드

## 4. 디자인 레버 (Design Levers)

### 4.1 리그/디비전 구조
| 레버 | 설명 |
|------|------|
| 승강 시스템 | 리그/디비전 구조, 승강 시스템 명확화 |
| 구현 방향 | 6단계 디비전, 100점 기준 승급/강등, 보호 매치 3판 |

### 4.2 시즌 메타 변화
| 레버 | 설명 |
|------|------|
| 테마 버프/너프 | 시즌마다 "핵심 테마 버프/너프" 부여로 메타 변화 유도 |
| 구현 방향 | 시즌1: 암살자 강화, 시즌2: 탱커 강화, 시즌3: 마법사 강화 |

### 4.3 카운터 설계
| 레버 | 설명 |
|------|------|
| 뚜렷한 카운터 | 카운터 빌드가 뚜렷이 존재하도록 스킬/역할 설계 |
| 구현 방향 | 암살자 ↔ 탱커 ↔ 마법사 삼각 상성, 스킬 상쇄 시스템 |

### 4.4 명예 시스템 분리
| 레버 | 설명 |
|------|------|
| 시즌 vs 누적 보상 | 시즌 종료 보상 + 누적 명예 시스템 분리 |
| 구현 방향 | 시즌 보상(스킨/자원), 누적 명예 포인트 → 명예 상점 교환 |

## 5. 재미 측정 지표 (KPI)

| 지표 | 목표 | 측정 방법 |
|------|------|-----------|
| 일일 매치 수 | 10회 이상 | 매치 로그 |
| 시즌 참여율 | 85% 이상 | 시즌 중 매치 유저 비율 |
| 디비전 분포 | 정규분포 | 각 디비전별 유저 비율 |
| 메타 다양성 | 상위 5빌드 점유 60% 이하 | 빌드 사용 통계 |
| D7 리텐션 | 30% 이상 | 7일차 잔존율 |
