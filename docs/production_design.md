## 1. 재미 요소 (Fun Factors)

### 1.1 핵심 재미 (Core Fun)
| 요소 | 설명 | 강도 |
|------|------|------|
| **PvP 경쟁** | 다른 플레이어와의 리그 경쟁 | ★★★★★ |
| **팀 빌딩** | 용병 조합과 시너지 구성 | ★★★★★ |
| **랭크 상승** | 브론즈부터 레전드까지 랭크업 | ★★★★☆ |
| **시즌 경쟁** | 시즌별 리셋과 보상 경쟁 | ★★★★☆ |

### 1.2 만족감 요소 (Satisfaction)
- **승리**: PvP 승리 시 트로피와 보상 획득
- **랭크업**: 티어 승급 시 특별 연출
- **시즌 보상**: 시즌 종료 시 등급별 보상
- **용병 획득**: 새로운 용병 획득 시 도감 등록

### 1.3 도전 요소 (Challenge)
```
도전 구조:
├── 리그 티어 시스템 (7단계)
├── 용병 상성 및 시너지
├── 메타 변화 대응
├── 시즌별 밸런스 패치
└── 토너먼트 경쟁
```

### 1.4 사회적 요소 (Social)
- **실시간 PvP**: 비동기 또는 실시간 대전
- **길드**: 길드 대전 및 협동
- **리더보드**: 글로벌/지역/친구 랭킹
- **관전**: 상위 랭커 경기 관전

## 2. 게임 루프 (Game Loop)

### 2.1 코어 루프 (3~5분)
```
┌─────────────────────────────────────────────────┐
│                    코어 루프                      │
├─────────────────────────────────────────────────┤
│  [상대 확인] → [팀 편성] → [배틀 시작]            │
│       ↑                            ↓             │
│       └──── [결과 확인] ← [오토배틀] ←┘          │
└─────────────────────────────────────────────────┘
```

**단계별 상세:**
1. **상대 확인** (5초): 매칭된 상대 정보 확인
2. **팀 편성** (30~60초): 상대에 맞는 용병 배치
3. **배틀 시작** (1초): 전투 개시
4. **오토배틀** (60~180초): 자동 전투 진행
5. **결과 확인** (5초): 승패 및 트로피 변동

### 2.2 세션 루프 (5~15분)
```
[접속] → [일일 보상] → [퀘스트 확인] → [리그 매칭] → [배틀]
                                           ↓
                              [용병 강화] → [팀 조정]
```

### 2.3 메타 루프 (장기)
```
┌──────────────────────────────────────────────────────┐
│                     메타 진행                         │
├──────────────────────────────────────────────────────┤
│  [배틀 승리] → [트로피 획득] → [리그 승급]            │
│         ↓                              ↓             │
│  [시즌 보상]                     [새 콘텐츠 해금]     │
│         ↓                              ↓             │
│  [용병 강화] ←───────────────────────────┘           │
└──────────────────────────────────────────────────────┘
```

### 2.4 플로우 다이어그램
```
[앱 시작]
    ↓
[메인 화면] ─────→ [용병 관리]
    │              [상점]
    │              [길드]
    │              [설정]
    ↓
[아레나] → [매칭] → [팀 편성] → [배틀] → [결과]
```

## 3. BM 모델 (Business Model)

### 3.1 수익화 전략
| 채널 | 비중 | 설명 |
|------|------|------|
| **시즌 패스** | 35% | 시즌별 프리미엄 보상 |
| **용병 가챠** | 30% | 신규 용병 획득 |
| **리워드 광고** | 20% | 티켓 충전, 보상 2배 |
| **편의 기능** | 15% | 에너지, 빠른 진행 |

### 3.2 과금 포인트
```
비과금 유저 경험:
├── 모든 리그 참여 가능
├── 기본 용병으로 경쟁 가능
├── 일일 무료 뽑기 제공
└── 시간 투자로 상위 랭크 도달 가능

과금 유저 혜택:
├── 시즌 패스 ($9.99/시즌)
├── 크리스탈 패키지 ($0.99~$99.99)
├── 한정 용병 번들 ($4.99~$19.99)
├── 스타터 팩 ($4.99)
└── 배틀 티켓 패키지 ($2.99)
```

### 3.3 가격 정책
| 상품 | 가격 | 가치 |
|------|------|------|
| 시즌 패스 | $9.99/시즌 | 전용 용병 + 보상 |
| 소량 크리스탈 (100) | $0.99 | 뽑기 1회 |
| 중량 크리스탈 (550) | $4.99 | 20% 보너스 |
| 대량 크리스탈 (1200) | $9.99 | 30% 보너스 |
| 스타터 팩 | $4.99 | SR 확정 + 크리스탈 |

### 3.4 리텐션 메커니즘
- **일일 보상**: 로그인 보너스
- **일일 퀘스트**: 배틀 3회, 승리 1회 등
- **시즌 시스템**: 30일 시즌, 랭크 리셋
- **시즌 패스**: 일일 미션으로 진행

## 4. 레벨 디자인 (Level Design)

### 4.1 난이도 곡선
```
상대 강도
  ▲
  │                    Legend ─
  │                Master ─╯
  │            Diamond ─╯
  │        Platinum ─╯
  │      Gold ─╯
  │    Silver ─╯
  │  Bronze ─╯
  └──────────────────────────────→ 트로피
    0   500  1000 1500 2000 2500
```

### 4.2 콘텐츠 진행
| 리그 | 트로피 | 보상/해금 |
|------|--------|----------|
| Bronze | 0~499 | 기본 보상, 튜토리얼 |
| Silver | 500~999 | 실버 팩, 길드 해금 |
| Gold | 1000~1499 | 골드 팩, 토너먼트 |
| Platinum | 1500~1999 | 플래티넘 팩 |
| Diamond | 2000~2499 | 다이아 팩 |
| Master | 2500~2999 | 마스터 전용 보상 |
| Legend | 3000+ | 레전드 전용 콘텐츠 |

### 4.3 밸런싱 공식
```dart
// 팀 전투력 계산
int calculateTeamPower(List<Mercenary> team) {
  return team.map((m) =>
    m.attack + m.defense + m.hp / 10 + m.skill.power
  ).sum.toInt() + getSynergyBonus(team);
}

// 매칭 점수
int matchmakingScore(Player player) {
  return player.trophies * 0.7 + player.teamPower * 0.3;
}

// 시즌 리셋
int softReset(int currentTrophies) {
  if (currentTrophies > 2000) {
    return 2000 + (currentTrophies - 2000) ~/ 2;
  }
  return currentTrophies;
}
```

### 4.4 보상 체계
| 보상 유형 | 획득 조건 | 수량 |
|-----------|----------|------|
| 승리 트로피 | PvP 승리 | +20~30 |
| 패배 트로피 | PvP 패배 | -10~20 |
| 승리 골드 | PvP 승리 | 50~200 |
| 시즌 보상 | 시즌 종료 | 티어별 |
| 시즌 패스 | 패스 레벨업 | 단계별 |

## 5. UI/UX 설계

### 5.1 정보 계층
```
레벨 1 (항상 표시):
├── 현재 트로피/리그
├── 골드/크리스탈
├── 배틀 티켓
└── 알림 배지

레벨 2 (상호작용 시):
├── 용병 상세 정보
├── 팀 편성 화면
├── 상대 정보
└── 배틀 진행

레벨 3 (팝업):
├── 가챠 결과
├── 랭크업 연출
├── 시즌 보상
└── 결과 화면
```

### 5.2 네비게이션 흐름
```
[메인 화면]
    ├── [아레나] → [매칭] → [배틀] → [결과]
    ├── [용병] → [용병 목록] → [강화/승급]
    ├── [뽑기] → [가챠 화면] → [결과]
    ├── [길드] → [길드 활동]
    ├── [상점] → [상품 목록]
    └── [설정] → [옵션]
```

### 5.3 피드백 시스템
| 이벤트 | 시각 | 청각 | 촉각 |
|--------|------|------|------|
| 스킬 발동 | 스킬 이펙트 | 스킬음 | 진동 (중) |
| 승리 | 승리 연출 | 승리 BGM | 진동 (중) |
| SSR 획득 | 황금 연출 | 특별 팡파레 | 진동 (강) |
| 랭크업 | 랭크 배너 | 랭크업 음악 | 진동 (강) |

### 5.4 접근성
- **자동 전투**: 기본적으로 자동 진행
- **배속 기능**: 1x, 2x, 3x
- **스킵 기능**: 결과만 보기
- **색맹 모드**: 속성 아이콘 패턴
