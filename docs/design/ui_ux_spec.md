# MG-0013: Arena Legends UI/UX 스펙

> **게임명**: Arena Legends (아레나 레전드)
> **장르**: PvP 배틀 RPG
> **카테고리**: Year 2 Core
> **문서 버전**: 1.0.0

---

## 1. 화면 설정

| 항목 | 값 |
|------|-----|
| **방향** | Portrait (세로) |
| **기준 해상도** | 1080 × 1920 |
| **지원 비율** | 16:9 ~ 21:9 |
| **타겟 FPS** | 60fps |

### 방향 선택 이유
- 팀 편성 UI 세로 최적화
- PvP 대전 양쪽 팀 배치
- 랭킹/리더보드 세로 스크롤

## 4. 터치 영역

| 동작 | 영역 | 제스처 |
|------|------|--------|
| 영웅 선택 | 72×72dp | 탭 |
| 영웅 배치 | 슬롯 ↔ 영웅 | 드래그 |
| 영웅 상세 | 영웅 위 | 롱프레스 |
| 전투 시작 | 버튼 | 탭 |
| 상대 검색 | 새로고침 | 탭 |

### 터치 피드백
- 영웅 선택: 프레임 하이라이트
- 배치 성공: 슬롯 인 애니메이션
- 전투 시작: 화면 전환 이펙트
